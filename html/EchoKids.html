Web UX/UI Designer Intern

From August 2023 to December 2023

Overview

Project Overview

EchoKids is a carefully curated education platform that facilitates connections between young musicians, families, and schools for lessons. As an intern at EchoKids, my responsibilities included enhancing the UX/UI of the website, conducting heuristic evaluations, enhancing the front-end experience, and optimizing SEO by integrating Mixpanel and Google Analytics.


Duration

Overall: 5 months.

From August 2022 to December 2022.

Project Goals


  • Revamp existing web pages and generate fresh mockups for the website, web application, and app.

  • Implement Google Tag Manager and Mixpanel for the website and web application.

  • Identify errors and bugs within the website and web application.

  • Perform five usability tests on the live website.

Tools

Wireframes & Mockups


As part of my responsibilities, I actively participated in the creation and ongoing refinement of wireframes and mockups for both the website and web application. This involved consistently updating and enhancing these visual representations to ensure a seamless user experience. Additionally, I undertook the task of developing dedicated phone versions of these mockups and wireframes, catering specifically to the needs and preferences of mobile users. By prioritizing mobile accessibility, I aimed to provide an optimized browsing experience across various devices.

Illustrations

I have created more than 10 unique and artistic illustrations using Canva, aiming to elevate the website's aesthetic charm, all the while ensuring alignment with the brand's visual identity.

Tags Implementation and Bugs

Implemented over 20 Google Tag Manager Tags for the Echokids website (Webflow).

Implemented over 20 Mixpanel Tags for the Echokids website application (Bubble).

Search and encounter more than 10 bugs in both the website and web application.

Usability Testing

I assemble a group of five individuals for the purpose of conducting research. Each participant will be required to sign up for an account, and the entire process will be meticulously recorded and comprehensively analyzed. The following provides the user criteria:


Parent with a child but just thinking about private lessons for them (child's age 7).


Parent with a child knowing which instrument they would like to play during a private lesson (child's age 7).


A student wanting to find a music teacher and knowing which instrument they would like to play (Age 15).


A user looking to apply as a mentor.


Schools looking to have music programs at their preschool.
